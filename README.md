# Household Budget Calculator

A Python-based household budget calculator that helps calculate total income, fixed expenses, variable expenses, loan costs and remaining monthly budget.

The program runs in the terminal and guides the user step by step through the budgeting process.

## Features
- Add multiple household members and incomes
- Calculates net income
- Handles fixed and variable expenses
- Supports housing loans with interest and amortization
- Supports additional loans
- Calculates total expenses and remaining monthly budget

## Tech stack
- Python

## How to run
1. Make sure Python is installed
2. Clone the repository
3. Run the program:

```bash
python main.py
